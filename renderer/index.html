👤 Account Management

📋 How to Get Facebook Cookies

  1. Install Cookie Editor extension in Chrome/Edge
  2. Go to facebook.com and log in
  3. Click the Cookie Editor icon → Click "Export" → "JSON"
  4. Copy the entire JSON array
  5. Paste into the "Cookies" field in the Add Account form

🎭 User Agent Setup

✅ Method 1: Auto-Generate (Recommended)
  1. Select device: Windows / Mac / Mac Mini / iPad
  2. Click "🎲 Generate UA"
  3. Done — unique UA generated automatically
Method 2: Manual Entry
  1. Check ☑ "Enter User Agent manually"
  2. Type or paste your own UA string
  3. Device type auto-detected

🌐 Proxy Format

Format: hostname:port:username:password
Example: proxy.example.com:8080:myuser:mypass
Leave empty to use your real IP

⚠️ Important Rules

  • Each account needs a unique User Agent (software prevents duplicates)
  • Use different proxies per account (residential proxies recommended)
  • Cookies expire every 7-14 days — update if automation stops working
  • Keep accounts enabled (green status) for automation to use them

📌 Post Management

Adding Posts

  1. Go to Posts tab → Click "+ Add Post"
  2. Select account to assign to
  3. Paste the full Facebook post URL
  4. Add optional title → Click "Save Post"

📥 Bulk Import

  1. Click "📋 Import Multiple"
  2. Select account → Paste URLs (one per line) → Click Import

✅ Supported URL Formats

✓ https://www.facebook.com/groups/123456/posts/789012/
✓ https://www.facebook.com/username/posts/123456789
✓ https://www.facebook.com/permalink.php?story_fbid=...

📂 Facebook Groups Mode

🤖 How Group Mode Works

  1. Navigates to each group URL you added
  2. Scrolls the feed to load recent posts
  3. Finds posts within a random 1-59 minute window
  4. Comments on up to 3 recent posts per group
  5. Rotates to next group with a delay

Adding Groups

  1. Go to Facebook Groups tab
  2. Click "+ Add Group" → Select account → Paste group URL → Save
  3. Or click "📋 Import Multiple Groups" for bulk import

⚙️ Enabling Group Mode

  1. Go to Settings → Automation Mode
  2. Select "📂 Facebook Groups Mode"
  3. Save Settings → Start Automation
⚡ Group Mode vs Post Mode:
  • Post Mode: You add specific post URLs — comments those exact posts
  • Group Mode: You add group URLs — auto-finds fresh posts and comments

💬 Comment Templates

Creating Templates

  1. Go to Templates tab → Select an account
  2. Click "+ Add Template" → Write your comment
  3. Add 3-5 templates for variety → Click "Save Templates"

🔄 How Automation Uses Templates

1. Post "Amazing!" (starter)

💡 Template Tips

  • Write natural, human-like sentences
  • Keep templates 10-50 words long
  • Make them generic enough to work on any post
  • Create 3-5 different templates per account for variety
  • Avoid promotional/spam language
EXAMPLE TEMPLATES:

"This is really helpful! Thanks for sharing."

"Great information! I was looking for exactly this."

"Awesome content! Keep posting more like this."

Final output: "Hi This is really helpful! Thanks for sharing. 👍"

⚙️ Settings Explained

💬 Comment Delay (seconds)

Wait time between comments. Recommended: 30-120 sec. Too fast = ban risk. Random between min/max = more human-like.

📊 Max Comments Per Account

Daily limit per account. Recommended: 20-30. Resets at midnight automatically.

🔄 Account Switch Delay

Wait before switching to next account. Recommended: 60-180 sec.

⏱️ Group Rotation Delay

Wait between groups in Group Mode. Recommended: 30-120 sec.

🗑️ Auto Delete Pending Comments

Automatically deletes comments stuck in "Pending" or "Declined" review. Keep Enabled.

👻 Headless Mode

Run browser invisibly. Faster but harder to debug. Not recommended for beginners.

⏰ Working Hours

Restrict automation to specific hours. Recommended: 9 AM – 9 PM. Makes behavior look more human.

🛡️ Anti-Detection & Security

🔒 WebRTC IP Leak Prevention

Completely disables WebRTC — your real IP never leaks even with proxy. Test at browserleaks.com/webrtc

🎨 Canvas Fingerprint Spoof

Adds random noise to canvas data. Each session has a different fingerprint — prevents cross-session tracking.

🔊 Audio Context Randomization

Randomizes audio fingerprint data to prevent audio-based tracking.

🎮 WebGL GPU Spoofing

Spoofs GPU as "Intel Iris OpenGL Engine" — all accounts appear as same generic GPU.

🤖 Bot Detection Prevention

Sets navigator.webdriver = false, removes automation indicators, disables AutomationControlled flag.

📍 Timezone & Geolocation

Sets timezone to match proxy location. Default: New York. Prevents location-based detection.

🎭 Behavioral Anti-Detection

  • Random Delays: Every action has randomized wait time
  • Human-like Typing: Types character-by-character (not instant paste)
  • Natural Scrolling: Smooth, random-speed scrolling
  • Account Cooldowns: Prevents over-activity per account
⚠️ Critical Security Rules
  • Always use unique proxies for each account
  • Never exceed 20-30 comments/account/day
  • Enable Working Hours restriction
  • Use different User Agents per account

❓ Frequently Asked Questions

Q: Is this software safe to use?

The software includes advanced anti-detection features, but automating Facebook always carries some risk. Follow best practices: use proxies, keep delays moderate, and don't exceed 20-30 comments/day per account.

Q: How many accounts can I run?

Technically unlimited, but 10-20 accounts is recommended. Each account needs a unique proxy and User Agent for safety.

Q: Can I use free proxies?

Not recommended. Free proxies are often banned by Facebook and unreliable. Use paid residential proxies for best results.

Q: How do I renew my license?

Contact the developer on Telegram: @Minhazulruhan1. Software stops working when expired.

Q: Can I transfer license to another PC?

License is hardware-locked. Contact developer for a transfer authorization.

Q: What if comments are always pending?

Enable "Auto Delete Pending Comments" in Settings. Also try groups with less strict moderation, or reduce comment frequency.

Q: Do cookies expire? How often should I update them?

Yes, Facebook cookies expire every 7-14 days. If automation stops working, re-login to Facebook and re-export cookies using Cookie Editor extension.
