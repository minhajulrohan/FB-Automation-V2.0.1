‚öôÔ∏è Settings Explained

Comment Settings:

  • Comment Delay: Time between comments (30-120 sec recommended)
  • Max Comments Per Account: Daily limit per account (20 recommended)
  • Account Switch Delay: Wait time before switching accounts
  • Post Rotation Delay: Wait time between different posts
  • Auto Delete Pending: Remove comments stuck in "pending" status

React Settings:

  • Enable Auto React: Automatically react after commenting
  • Reaction Types: Choose which reactions to use (Like, Love, etc)
  • Reaction Probability: Chance of reacting (70% recommended)
  • Reaction Delay: Wait time before reacting (2-8 sec)

Browser Settings:

  • Headless Mode: Run browser invisibly (faster but can't monitor) but not Recomand

Working Hours:

  • Enable Restriction: Only run during specific hours
  • Hours: Set start and end time (e.g., 9 AM - 9 PM)
